
--For Discord Overlay--

使用方法

	①フォルダ内の ForDiscordOverlay を起動してください。
	②Discordの設定のゲームアクティビティからゲームの追加でこのアプリを選んでください。
	※うまく認識されえない場合は解像度設定画面で追加してみてください。
	③あとはオーバーレイを有効にしたらオーバーレイが使えるようになります。

	※このアプリを起動中にDiscordのゲームアクティビティを再度開くとアプリが実行していない判定になってしまうため、注意してください。
	※その場合アプリを再起動してください。

	・その他不具合を多数確認していますが技術的な問題や、致命的ではないという判断の結果、放置させていただいております。
